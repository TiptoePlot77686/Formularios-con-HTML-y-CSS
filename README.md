# Formulario de registro de usuario — Reto 3 UVEG

Un formulario en HTML y CSS con campos de Nombre, Apellido, Correo, Contraseña, descripción breve y selección de intereses con íconos (Font Awesome). Diseño claro, accesible y chill.

## Estructura del proyecto

- `index.html`: estructura del formulario
- `styles.css`: estilos visuales aplicados

## Secciones HTML

- `<h1>`: título principal
- `<input>`: campos de texto (nombre, apellido, correo, contraseña)
- `<textarea>`: descripción personal
- `<fieldset>` + `checkboxes`: intereses con íconos
- `<button>`: envío del formulario
- `<a>`: enlace para usuarios registrados

## Estilos CSS

- Tipografía: Segoe UI
- Colores: azul medio, gris claro, blanco
- Bordes redondeados y efectos `:focus`
- Distribución con flexbox
- Chips de intereses con íconos Font Awesome
